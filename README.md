## Task 4 Completed – AI Chatbot

I have successfully completed **Task 4** of my APSCHE internship by developing an **AI Chatbot**.

### Project Overview

This project is an AI-based chatbot that interacts with users by answering questions related to artificial intelligence. It provides predefined responses for AI-related queries and simulates basic conversational interaction.

### Features Implemented

* User question input
* AI-related question answering
* Interactive chatbot conversation
* Multiple predefined responses
* Exit option to end chat
* Simple and easy-to-use interface
* Real-time response generation

### Technologies Used

* Python
* Conditional statements
* Functions
* Input/output operations

### Outcome

This task helped me improve my understanding of chatbot development, Python programming, decision-making logic, and user interaction design.
